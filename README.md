## Завдання 2: Обчислення інтеграла методом Монте-Карло

### Опис

Обчислення визначеного інтеграла функції f(x) = x² на відрізку [0, 2] методом Монте-Карло з використанням 200 000 випадкових точок.

### Аналітичне розв'язання

Для функції f(x) = x² інтеграл обчислюється за формулою:

$$\int_0^2 x^2 \, dx = \left[ \frac{x^3}{3} \right]_0^2 = \frac{2^3}{3} - \frac{0^3}{3} = \frac{8}{3} \approx 2.6667$$

### Результати обчислень

| Метод                      | Результат                | Похибка                    |
|----------------------------|--------------------------|----------------------------|
| **Монте-Карло** (n=200000) | 2.66688                  | —                          |
| **Аналітичний**            | 2.6666666666666665       | —                          |
| **scipy.integrate.quad**   | 2.6666666666666665       | ≈ 2.96 × 10⁻¹⁴             |

### Порівняння результатів

| Порівняння                        | Абсолютна різниця        |
|-----------------------------------|--------------------------|
| Монте-Карло vs Аналітичний        | 0.000213                 |
| Монте-Карло vs quad               | 0.000213                 |
| Аналітичний vs quad               | ≈ 0 (машинна точність)   |

## Висновки

### 1. Правильність методу Монте-Карло

Метод Монте-Карло дає **коректний результат**, що підтверджується порівнянням з еталонними значеннями:

- **Аналітичне значення** інтеграла ∫₀² x² dx = 8/3 ≈ 2.6667
- **Результат Монте-Карло**: 2.66688
- **Результат scipy.integrate.quad**: 2.6666666666666665

Абсолютна похибка методу Монте-Карло становить лише **0.000213** (або ~0.008%), що свідчить про високу точність оцінки при використанні 200 000 випадкових точок.

### 2. Порівняння з scipy.integrate.quad

Функція `quad` із бібліотеки SciPy використовує адаптивні квадратурні методи (алгоритм QUADPACK) і забезпечує:
- Результат, що збігається з аналітичним до машинної точності
- Оцінку абсолютної похибки ≈ 2.96 × 10⁻¹⁴

На відміну від `quad`, метод Монте-Карло є **стохастичним** і його точність залежить від кількості випадкових точок. Збільшення кількості точок зменшує похибку пропорційно √n.

### 3. Переваги та недоліки методу Монте-Карло

| Переваги                                        | Недоліки                                      |
|-------------------------------------------------|-----------------------------------------------|
| Простота реалізації                             | Нижча точність порівняно з quad               |
| Легко застосовується до багатовимірних інтегралів | Повільна збіжність (O(1/√n))                |
| Не потребує знання первісної функції            | Результат є випадковою величиною             |

### 4. Загальний висновок

Реалізований метод Монте-Карло **працює правильно** і дає результати, що добре узгоджуються як з аналітичним обчисленням, так і з результатами функції `scipy.integrate.quad`. Для практичних застосувань, де потрібна висока точність одновимірних інтегралів, рекомендується використовувати `quad`. Метод Монте-Карло є особливо корисним для багатовимірних інтегралів або випадків, коли підінтегральна функція має складну структуру.
